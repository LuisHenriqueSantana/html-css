Principais Formatações

Negrito / Demantica

Nessa frase temos um termo em negrito usando a tag <b> (não semântica)

nesta frase tenho um termo em destaque utilizando a tag <Strong>

Italico / Enfase

Nesta frase temos um termo em Italico utilizando a tag <I> (Não semântica)

Nesta Frase temos um termo em Ênfase usando a tag <EM> (semântica)

Texto Marcado

podemos criar támbem um texto marcado usando a tag <MARK>

e no outro parágrafo temos outro textos marcado no final.

Texto Grande e Pequeno

Estamos criando um texto grande e um pequeno. (<BIG>.)(<SMALL>; semântica)

Texto Deletado

Podemos marcar um texto para que ele seja lido mas não considerado usamos a tag <DEL>.

Texto Inserido

Podemos marcar um texto como inserido para adicionar uma ênfase e indicar que ele foi adicionado depois com a tag <INS>.

Existe também o sublinhado com a tag <U>(Não semântica).

Texto Sobrescrito

Para inserir coisa do tipo x20+3 use a tag <sup>

Texto Subscrito

Para inserir coisa tipo H2O use a tag <sub>
